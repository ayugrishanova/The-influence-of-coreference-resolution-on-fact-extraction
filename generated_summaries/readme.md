# Структура директории

Всего в работе использовались два метода: baseline и QF-MSD. Каждый из них тестировался на следующих коллекциях:

- Коллекция без разрешения кореференции
- Коллекция текстов с разрешенной кореференцией (+coref)
- Сдвоенная коллекция как оригинальных текстов, так и текстов с разрешенной кореференцией (+coref+orig)
